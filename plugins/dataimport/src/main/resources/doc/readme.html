Data Import

Thorsten Mürell

The data import plugin allows to import arbitary files into existing tables in the database. Currently the following types of files are supported:

  • Microsoft Excel files
  • Comma separated files (CSV)

Both file formats offer you some selections to choose the sheet (Excel) or the seperator and file encoding (CSV).

To import a file, right click a table and choose "Import file" from the menu.

After that you can map the table's columns to the colums of the import file. At the top of dialog you firstly choose, if the first row contains the column headers. Then you can associate the existing table columns with the import file columns. You also have several "special" options:

  • Skip: This column is not used. The default value of the database will be used.
  • NULL: This column will be set to NULL.
  • Fixed value: This column will be set to fixed value, that you can specify in the third column named "Fixed Value".
  • Auto increment: This column will be incremented by one each insert. You can specify the start value by typing a number into the "Fixed Value" column.
