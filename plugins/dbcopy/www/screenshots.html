Right-click on selected tables in the source database to access the "Copy Table" menu.
Right-click on selected destination schema to access the "Paste Table" menu.
Confirm record deletion if destination tables already have records.
Watch the progress bar indicate how the copy is going.
Cancel the copy operation at any time.
A dialog will let you know how much time was spent copying.
Copy operation can be optimized using the Global Preferences sheet in SQuirreL.
