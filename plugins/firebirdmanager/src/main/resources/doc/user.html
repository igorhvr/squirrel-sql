Firebird Manager Plugin
- User manager -



With this module you can manage firebird users.

To manage a database user, follow these steps:



1. At first you must fill in the connection parameters for the database server, whose users should be managed.
 Set the server, port, username (with sufficient permissions) and his password.
 Then active the connection checkbox.

2. If all input is correct, you will see the stored users of this server in the table thereunder.
 Now you can create a new user or edit one, which must be selected.

3. If you create a new one, username, password and confirm password are mandatory;
 When you edit a user, you can leave the password and confirm password blank,
 if you prefer to use the old password.
