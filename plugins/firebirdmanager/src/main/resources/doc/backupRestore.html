Firebird Manager Plugin
- Backup and restore -


With this module you can backup and restore a firebird database.

To backup a database, follow these steps:



1. Set a valid username and password with sufficient permissions.
 If you want to display the backup process, you must check display process.
 The process will be displayed in the text area of this window.

2. Set the database server and port.
 When your server is a localhost, you can choose the database and backup file with the file browser.
 Otherwise you must type in the right folder and file name yourself.

3. If you want to use a backup file with a timestamp,
 you can generate one with the button after the input field.

4. Choose some of the backup options for the firebird database.

5. Now start up the backup and see the process in the text area below, if display process is checked.

6. You can save these options for future reference.


To restore a database, follow these steps:



1. Set a valid username and password with sufficient permissions.
 If you want to display the restore process, you must check display process.
 The process will be displayed in the text area of this window.

2. Set the database server and port.
 When your server is a localhost, you can choose the database and backup file with the file browser.
 Otherwise you must type in the right folder and file name yourself.

3. You can also get the file names from the backup page. Use the first button after the input fields.

4. Choose some of the restore options for the firebird database.
 The field new page size will only be ensabled, when override old page size is selected.

5. Start up the restore and see the process in the text area below, if display process is checked.

6. You can save these options for future reference.
