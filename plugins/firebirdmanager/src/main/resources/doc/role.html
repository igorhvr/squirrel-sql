Firebird Manager Plugin
- Role manager -



With this module you can manage firebird roles.

To manage a database role, follow these steps:



1. To create a new role, click on the new button, enter the role name and click save.
2. The owner of the role is the currently registered user.

Important notice:
 You cannot edit a role! If you want to change the rolename you must delete the role and create it new.
