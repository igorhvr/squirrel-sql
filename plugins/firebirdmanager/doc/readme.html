Firebird Manager Plugin

Author


Introduction

The Firebird Manager plugin adds the ability to manage a firebird server and database.
This plugin needs the firebird jdbc driver jaybird www.firebirdsql.org/index.php?op=files&id=jaybird
Version 2.0 or greater (special thanks to the jaybird team)

It includes the following modules:

  • Backup and restore manager (callable from 1 and 2)
  • Create database manager (callable from 1 and 2)
  • User manager (callable from 1 and 2)
  • Role manager (callable from 2)
  • Grant and revoke manager (callable from 2)
You can call the first 3 modules from the main menu entry Plugins. They can also be used by opening the session popup menu. The Role manager and Grant manager can only be started from the session popup menu (see also the pictures below).

Options can be configured on the own global preferences tab.
