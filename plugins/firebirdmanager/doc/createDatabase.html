Firebird Manager Plugin
- Create database -



With this module you can create an empty firebird database.

To create a database, follow these steps:



1. Choose the target folder and the database file name.

2. Then set the database server address or name and the server port.
 Type in a valid database user and his password, which shall be the owner of the new database.

3. Select the sql dialect.

4. Now you can start the database creation.

5. You can save these options for future reference.
