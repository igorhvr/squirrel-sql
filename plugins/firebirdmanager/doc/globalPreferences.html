Firebird Manager Plugin
- Global preferences -



On the tab you can set the default values for some of the input fields for the Firebird Manager Plugin.



1. You can find this tab in the following menu entries:
 -> main menu -> global options and then the tab firebird manager.

2. You can set the following default values:
  • server: an ip or hostname of your standard database server
  • port: the port of your default server
  • database folder: the default database folder of your server
  • properties folder: the default folder to save your user properties (memorized options) of the modules
  • username: the default username of your server
3. OK to save the options, Close to exit the window.
