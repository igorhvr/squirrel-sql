Introduction

The Oracle plugin adds Oracle specific functionality to SQuirreL SQL Client. Read access is required to the following system views in order for this additional functionality to work correctly.

  • sys.all_source
  • sys.all_sequences
  • sys.all_trigger_cols
  • sys.all_triggers
  • sys.user_users
  • sys.v_$instance
  • sys.v_$option
  • sys.v_$session
  • sys.v_$sesstat
  • sys.v_$statname

New Tabs

The session node in the object tree has a new "Options"tab. This tab displays the Oracle server options.

Stored Procedures now have a "Source" tab which displays the source of the stored procedure.

New Object Types

The session node in the object tree has the new object type nodes:

USERS, INSTANCES and SESSIONS.

New object types have been added to the schema nodes in the object tree. These include:

Package, Consumer Group, Function, Index, LOB, Sequence, and Type.
