Smarttools Plugin
- Search null values in NOT NULL columns -



With this module you can find all NOT NULL columns in the database with a null value.

To find these bad values, follow these steps:



1. Insert the table name, which columns shall be searched. You can use a % as wildcard.
 An empty input field means, that all tables will be used.

2. Start for searching these bad columns.
 You can terminate searching by clicking the stop button.

3. The result will be displayed in the table.

4. If there is a result displayed in the table, the print button will be activated to print the result.
