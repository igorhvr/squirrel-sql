Introduction

This plugin allows you to export the Database Driver definitions, the Database Alias definitions and the Application Settings to files on the file system. This lets you setup your defintions once and put them on as many other PCs as you want to.

In order to use these files on another PC they need to be called SQLDrivers.xml (Database Driver definitions), SQLAliases.xml (Database Alias definitions), and prefs.xml (Application Settings) and placed in the directory .squirrel-sql within the users home directory.

New Menu Options

A new hierarchical menu "Export" has been added to the Plugins menu. After taking one of the export options you will be prompted for a directory to export to.
