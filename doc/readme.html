Introduction

SQuirreL SQL Client is a graphical Java program that will allow you to view the structure of a JDBC compliant database, browse the data in tables, issue SQL commands etc.

It is free software, and you are welcome to redistribute it under the terms of the GNU Lesser General Public License.

From version 1.1 on SQuirreL SQL Client requires JDK1.3 to work. If you need a version compatible with JDK1.2 download SquirreL SQL Client 1.0.
