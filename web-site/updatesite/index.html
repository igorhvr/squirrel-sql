SQuirreL SQL Client Update Site


This is a site that provides updated versions of SQuirreL SQL Client software artifacts.
SQuirreL has a feature that allows the user to update to the latest version that is available,
without having to re-install. This site exists solely to service that feature.

If you were not looking for updates to existing installations of SQuirreL, then perhaps you
intended to go to the project website, which is hosted on SourceForge.net
